# Onboarding Guide

## Goal
Get AGenNext Kernel running locally and execute your first governed agent task.

## Steps

1. Clone repo
2. Setup .env
3. Run docker compose
4. Open UI
5. Execute first task via API

## What to Expect

- Policy enforcement in action
- Logged decisions
- UI telemetry

## Common Issues

- Docker not running
- Port conflicts
- Missing env variables
